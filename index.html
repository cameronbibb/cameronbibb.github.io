WILLOW

Willow is an open-source framework for creating change data capture (CDC) pipelines built to connect existing data stores and update caches in near real-time. Built using Node.js, Debezium, and Kafka, Willow is easy to set up with your existing PostgreSQL database.

Willow was built to solve the cache consistency problem, while also making it easy for developers to set up a CDC pipeline, connecting a source database with a target cache.
